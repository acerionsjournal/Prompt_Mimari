Kurulum:
Bu program kurulum gerektirmez. Doğrudan çalıştırabilirsiniz.

Kullanım:
1. "AI_Prompt_Mimari.exe" dosyasına çift tıklayın.
2. Program otomatik olarak tarayıcınızda açılacaktır.
3. Kullanım bittiğinde programı kapatmak için "AI_Prompt_Mimarı_Kapat.bat" dosyasına çift tıklayın.

Not:
Eğer program açılmazsa, bilgisayarınızda "Visual C++ Redistributable" paketinin yüklü olduğundan emin olun. Genellikle çoğu bilgisayarda yüklüdür.

İyi çalışmalar!
